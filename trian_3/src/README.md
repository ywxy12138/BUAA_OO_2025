总共有两个bug。  
在通过初始运行`TestRunner`代码后发现对于`delete`操作的测试点未能正常进行，而对于`insert`操作的测试点只进行了一次插入后就报错了。  
因此，初步将错误定位在`delete`和`insert`两个函数上，通过观察并结合对于测试结果，发现`delete`函数中，二分查找法的循环条件并没有`left==right`的情况，也就是当左右边界恰好相等且刚好是要删除的数的时候无法更新相应坐标，会导致无法正确删除的情况发生。而在`insert`函数中，则是左边界的起点初始化错成了1，而不是从真正的左边界0下标开始，会导致数组内元素只有一个的时候，无法进行正常的插入操作。  
修改过后，就可以顺利通过单元测试了！！！
